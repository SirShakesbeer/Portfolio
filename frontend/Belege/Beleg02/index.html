Mobile-First - Navbars für Handys

Die Navbar auf dieser Seite verbindet mehrere Funktionalitäten zu Einer. Durch die navbar-Klasse von Bootstrap werden die Items schonmal richtig formatiert. Ein Button, der nur bei kleineren Displays gezeigt wird, kann die Anzeige der Navbar togglen. Ein Scrollspy im body-Element sorgt dafür, dass die Bar immer anzeigt in welcher Sektion man sich gerade befindet und durch eine eigene ID wird die Position so gesetzt, dass die Navbar beim Scrollen mitkommt.

Modalität

Modals sind ein gutes Werkzeug um Benachrichtigungen oder Hinweise anzuzeigen. Hier benutze ich sie für die Anzeige meiner Kontaktdaten um die Seite etwas sauberer zu halten.

  • Modals are built with HTML, CSS, and JavaScript. They’re positioned over everything else in the document and remove scroll from the so that modal content scrolls instead.
  • Clicking on the modal “backdrop” will automatically close the modal.
  • Bootstrap only supports one modal window at a time. Nested modals aren’t supported as we believe them to be poor user experiences.

- getbootstrap.com

Die dunkle Seite

Komm auf die dunkle Seite (der Macht)! Im Ernst kommen Sie rüber! Worauf warten Sie? Ess ist sehr gemütlich hier. Wollen Sie oder nicht? Es lohnt sich. Sie verpassen was!

Dunkel ist es doch ganz gut oder? Der Effekt wurde durch das anwenden des Attributs data-bs-theme=dark/light auf das HTML Element erreicht. In der CSS-Datei kannn man dann die Bootstrap-Grundeinstellungen für die jeweiligen Themes überchriben, wodurch das Theme auch weiterhin so ein interessantes grün behalten kann.

Arduino-Schmarduino

Im ersten Semester der Studiengänge Informatik und Medieninformatik nimmt man laut Plan am Modul Digitaltechnik I teil. In zwei Praktika muss man im Hardware-Labor verschiedene Aufgaben bewältigen. Die erste Aufgabe hat ein wenig mit Software-Hardware Interaktion und Elektrotechnik zutun. Im zweiten Praktikum soll eine Lichtshow für einen 30 Sekunden langen Song-Ausschnitt auf einem Arduino programmiert werden. Beide Praktika wurden in Gruppen bearbeitet und haben sogar ziemlich Spaß gemacht.

Im Informatik Studiegang gibt es auch im zweiten Semester Aufgaben im Hardware-Labor. Für Medieninformatik ist das allerdings nicht der Fall. Ich musste diese Lücke also selbst ein wenig schließen und habe seitdem immer mal ein paar eigene Projekte mit Arduino ausprobiert.

Mit Arduino (oder MicroPython) lässt sich ziemlich viel lustiges Spielzeug bauen.

Bilder-Display

Mein erstes privates Projekt war ein kleines Display, welches Bilder (eine spätere Version auch GIFs) darstellen sollte. Dafür habe ich einen Seeed Studio XIAO RP2040 verwendet. Dieser ist wortwörtlich so groß wie ein Thumbnail und somit gut für kleine Projekte geeignet. Als Display habe ich ein simples (und günstiges) LCD-Display verwendet. Als erstes Projekt bereitete mir dieses Projekt alleine schon neue Herausforderungen, wodurch ich viel lernen konnte. Der Umgang mit der Arduino IDE und den Libraries für die verschiedenen Hardware-Teile sowie Löten wurden etwas geübt.

WiFi-Thermometer

Für ein anderes Projekt benutzte ich die PlatformIO IDE. Dies ist ein Plugin für Visual Studio Code mit allen wichtigen Funktionalitätn von Arduino. Dieses Projekt sollte ein Geschenk für ein Familienmitglied sein. Auf einem existierenden Display werden in der Küche Außen und Innentemperatur angezeigt. Mein Projekt sollte jetzt die Pool-Temperatur auf dem gleichen Display anzeigen. Dafür verwendete ich ein DS18B20-Sensor, da dieser wasserdicht ist und ein langess Kabel besitzt, was den Außeneinsatz etwas erträglicher macht. Der Chip war dieses mal ein WeMos S2 Mini. Er basiert ganz klasssisch auf dem ESP32-Chip. Das wichtigste Feature war allerdings die WIFI Funktionalität. Nach passendem Zusammenlöten und einem kleinen USB-Port-Problem (welches unter Windows nach dem Upload entsteht und sehr viel Zeit in Anspruch nahm), konnte das Problem zum ersten mal laufen. Momentan wird die Temperatur nur auf einer eigenen Website dargestellt, die Interaktion mit bestehenden Servern isst noch in Arbeit.
