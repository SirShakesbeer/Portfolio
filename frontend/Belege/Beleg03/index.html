Visual Storytelling im Web

"Ein Bild sagt mehr als tausend Worte."

(Redewendung, unbekannte Herkunft)

Definition

"Visual Storytelling ist die Kunst, eine Geschichte mithilfe visueller Elemente wie Bilder, Grafiken, Videos und Animationen zu erzählen.
Ziel ist es, komplexe Inhalte auf emotionale Weise zu vermitteln und dem Publikum
durch ansprechende Bilder und klare Strukturen den Kern der Botschaft näherzubringen."

[Q1]

Warum visual Storytelling?

* Besucher auf der Seite behalten
* Produkte einprägend präsentieren [Q2]
Picture Superiority Effect:
* Menschen merken sich Bilder besser als Text [Q3]

Prinizpien des Visual Storytelling

📌 1. Emotionen wecken

  • "The more emotionally invested you are in anything in your life,
    the less critical and the less objectively observant you become." [Q4]

📌 2. Struktur & Orientierung schaffen

  • Gesetz der Nähe, Ähnlichkeit, Einfachheit, Fortsetzung und Geschlossenheit nutzen [Q5]

📌 3. Interaktion mit dem Nutzer

  • Videos, Übergänge und Animationen machen Websites ansprechender

Tools

D3.js

= data driven documents

  • JavaScript-Bibliothek für interaktive und datenbasierte Visualisierungen
  • Nutzt SVG, HTML und CSS, um Daten dynamisch darzustellen
  • ermöglicht animierte und interaktive Diagramme

Offizielle Website: D3.js
Simplere Alternative: Plot von Observable



Three.js

  • JavaScript-Bibliothek für 3D-Grafiken und Animationen im Browser
  • Ermöglicht immersive Erlebnisse durch WebGL
  • z.B. durch Verknüpfung mit Scroll-Effekten

Offizielle Website: Three.js

ScrollMagic

  • JavaScript-Bibliothek, um Scroll-basierte Animationen zu erstellen
  • gut für Parallax-Effekte
  • Beispiel

Offizielle Website: ScrollMagic

Lottie Files

  • Open-Source-Tool für animierte SVGs und JSON-basierte Animationen
  • Konvertiert After Effects-Animationen für das Web

Offizielle Website: LottieFiles

Mapbox

  • API für interaktive Karten und geobasierte Visualisierungen
  • Unterstützt Storytelling durch ortsbezogene Informationen
  • Anpassbare Karten mit Marker-Clustering und Animation
  • auch interessant: StoryMap - für historische, geobasierte Infos

Offizielle Website: Mapbox



Quelle: mapbox, 2025. Mehr: interactive map

GSAP

= GreenSock Animation Platform

  • Animationsbibliothek für komplexe, flüssige Web-Animationen
  • Unterstützt Timeline-basierte Animationen und Interaktivität
  • Perfekt für Mikrointeraktionen und dynamische Webseiten-Elemente

Offizielle Website: GSAP

Live-Demo

Mit GSAP lassen sich simple Animationn bauen.

gsap.timeline({ repeat: -1 })
.to(".box", { duration: 1, x: 200, ease: "power2.inOut" }) // Nach rechts bewegen
.to(".box", { duration: 1, y: 200, ease: "power2.inOut" }) // Nach unten bewegen
.to(".box", { duration: 1, x: 0, ease: "power2.inOut" }) // Nach links bewegen
.to(".box", { duration: 1, y: 0, ease: "power2.inOut" }); // Nach oben bewegen



gsap.timeline({ repeat: -1 })
.to(".box", { stagger:1, duration: 1, x: 200, ease: "power2.inOut" }) // Nach rechts bewegen
.to(".box", { stagger:1, duration: 1, y: 200, ease: "power2.inOut" }) // Nach unten bewegen
.to(".box", { stagger:1, duration: 1, x: 0, ease: "power2.inOut" }) // Nach links bewegen
.to(".box", { stagger:1, duration: 1, y: 0, ease: "power2.inOut" }); // Nach oben bewegen
